# n8n — แจ้งเตือนอีเมล HR-Lite

HR-Lite **ไม่ส่งอีเมลจาก .NET โดยตรง** — API โพสต์ JSON ไปที่ n8n webhook แล้วให้ workflow ส่งเมล (SMTP / Gmail ฯลฯ)

## เชื่อมต่อให้ครบ (ครั้งแรก)

1. **รัน n8n**
   ```bash
   make n8n-up
   ```
   หรือ `make dev` (จะรัน `db-up` + `n8n-up` ให้อัตโนมัติ)

2. **นำเข้า workflow**
   - เปิด http://localhost:5678
   - **Workflows** → **Import from file** → เลือก [`hr-lite-events.workflow.json`](./hr-lite-events.workflow.json)
   - เปิด workflow → **Activate** (สวิตช์มุมขวาบน — ต้อง Active ถึงจะรับที่ `/webhook/hr-events`)

3. **SMTP (ถ้าต้องการส่งเมลจริง)**
   - ใน node **Send Email** → Credentials → ตั้ง SMTP (Gmail app password, Mailtrap ฯลฯ)
   - ไม่ตั้ง SMTP ก็ยังเห็น execution ใน n8n ได้

4. **ทดสอบจากเทอร์มินัล**
   ```bash
   make n8n-test
   ```
   หรือจาก API (ต้อง login เป็น Hr/Admin, Development เท่านั้น):
   ```http
   POST http://localhost:5100/api/notifications/test-n8n
   Authorization: Bearer <token>
   ```

5. **ทดสอบจากแอป**
   - Login เป็น `emp001` / `password123` → ยื่นคำขอลา
   - Manager (`mgr.eng`) จะได้อีเมลที่ `wichai.j@hr-lite.local` (ถ้า SMTP พร้อม)

## URL และ config

| สภาพแวดล้อม | Webhook URL |
|-------------|-------------|
| Local `make dev` | `http://localhost:5678/webhook/hr-events` |
| Docker stack | `http://n8n:5678/webhook/hr-events` (ใน `docker-compose.yml`) |

ค่าใน [`appsettings.Development.json`](../../src/NexusCore.Api/appsettings.Development.json):

```json
"N8n": {
  "WebhookUrl": "http://localhost:5678/webhook/hr-events",
  "WebhookSecret": "dev-secret"
}
```

## Payload

API ส่ง JSON แบบ flat (ไม่ห่อใน `body`):

```json
{
  "eventType": "leave.submitted",
  "recipientEmail": "manager@example.com",
  "subject": "คำขอลาใหม่รออนุมัติ",
  "body": "สมหญิง ยื่นคำขอลา 1–3 มิ.ย.",
  "metadata": { "userId": "...", "linkPath": "/leave/..." }
}
```

Workflow มี node **Normalize Payload** รองรับทั้งรูปแบบ flat และ `$json.body.*` จาก webhook ของ n8n

## Event types

| eventType | เมื่อไหร่ |
|-----------|----------|
| `leave.submitted` | ส่งคำขอลา |
| `leave.approved` | อนุมัติลา |
| `leave.rejected` | ปฏิเสธลา |
| `leave.cancelled` | ยกเลิกลา |
| `overtime.submitted` / `approved` / `rejected` | OT |
| `expense.submitted` / `decided` | เบิกค่าใช้จ่าย |
| `announcement.published` | ประกาศ HR |
| `payslip.published` | อัปโหลดสลิป |
| `onboarding.assigned` | มอบหมาย onboarding |
| `review.opened` | เปิดรอบประเมิน |

## แก้ปัญหา

| อาการ | แก้ |
|--------|-----|
| `404` จาก webhook | Workflow ยังไม่ **Activate** |
| `Cannot reach n8n` | รัน `make n8n-up` |
| เห็น execution แต่ไม่มีเมล | ตั้ง SMTP ที่ node Send Email |
| `make n8n-test` ล้มเหลว | เปิด n8n UI → ตรวจว่า workflow Active และ path = `hr-events` |
